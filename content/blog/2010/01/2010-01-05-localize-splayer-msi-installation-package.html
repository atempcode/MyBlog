{% extends "_post.html" %} {% hyde title: Localize SPlayer MSI Installation Package created: 2010-01-05 21:49:11 %} {% block article %} Current SPlayer MSI supports English, Simplified Chinese, Traditional Chinese and Russian. To add a new language: Check if the languiage is supported in Wix 3. these languages are supported: cs-cz, de-de, en-us, es-es, fr-fr, hu-hu, it-it, ja-jp, nl-nl, pl-pl, ru-ru, uk-ua. if not, you need to create your own translation of the UI strings. Check the svn://svn.splayer.org:3690/svplayer/Setup/svplayer.msi/WiX/zh-cn.wxl for reference. Add a configuration in SvPlayer.wixproj. Pay attention to the Culture, locale and codepage. (You can get the information from MSDN:[http://msdn.microsoft.com/en-us/goglobal/bb896001.aspx](http://msdn.microsoft.com/en-us/goglobal/bb896001.aspx) ) Take the Russion for example: {% syntax xml %} locale=1049;codepage=1251 ru-RU bin\$(Configuration)\ obj\$(Platform)\$(Configuration)\ {% endsyntax %} Change the WiX\build.bat to build the MSI fro the new language. Still take Russian for example: > msbuild svplayer.wixproj /property:Configuration=ReleaseRU /l:FileLogger,Microsoft.Build.Engine;logfile=svplayer.ru.log;verbosity=detailed /t:Clean,Build Create the transform: update the build.bat file under transform folder: ... msitran -g ..\wix\bin\Release\en-US\SvPlayer.msi ..\wix\bin\ReleaseRU\ru-RU\SvPlayer.msi ru.mst ... scripts\wisubstg.vbs ..\..\..\..\SPlayer.msi ru.mst 1049 scripts\WiLangId.vbs ..\..\..\..\SPlayer.msi Package 1033,2052,1028,1049 Reference: [http://trac.splayer.org/changeset/1071](http://trac.splayer.org/ch angeset/1071) This changeset adds the Russian support.
