Actually makeinfo is installed and the version is 4.11 **Analysis:** The configure script of binutils-2.18 check the version by: ` # For an installed makeinfo, we require it to be from texinfo 4.4 or # higher, else we use the "missing" dummy. if ${MAKEINFO} --version \ | egrep 'texinfo[^0-9]*([1-3][0-9]|4\.[4-9]|[5-9])' >/dev/null 2>&1; then : else MAKEINFO="$MISSING makeinfo" ` So it thinks 4.11 as 4.1. **Patch it** ` RCS file: /cvs/src/src/configure,v retrieving revision 1.277 retrieving revision 1.278 diff -u -r1.277 -r1.278 --- src/configure 2007/09/14 14:51:36 1.277 +++ src/configure 2007/09/14 15:47:01 1.278 @@ -6192,7 +6192,7 @@ # For an installed makeinfo, we require it to be from texinfo 4.4 or # higher, else we use the "missing" dummy. if ${MAKEINFO} --version \ - | egrep 'texinfo[^0-9]*([1-3][0-9]|4\.[4-9]|[5-9])' >/dev/null 2>&1; then + | egrep 'texinfo[^0-9]*(4\.([4-9]|[1-9][0-9])|[5-9]|[1-9][0-9])' >/dev/null 2>&1; then : else MAKEINFO="$MISSING makeinfo"` {% endblock %}
